Trong giao diện chỉnh sửa (nano):
Tìm dòng: const API_URL = 'http://localhost:4000/api';
Sửa thành: const API_URL = 'http://203.145.47.207:4000/api'; (IP server bạn đã cung cấp).
Bấm Ctrl+O, Enter để lưu, và Ctrl+X để thoát.

# Cài đặt thư viện (nếu chưa cài)
npm install

# Đóng gói dự án (kết quả sẽ nằm trong thư mục 'dist')
npm run build


# Cài đặt serve (công cụ chạy web tĩnh) nếu chưa có
sudo npm install -g serve

# Chạy web ở cổng 5173 bằng PM2 (để nó không tắt khi bạn thoát VPS)
pm2 start "npx serve -s dist -l 5173" --name lms-frontend

# Lưu lại trạng thái PM2 để tự khởi động lại nếu VPS reset
pm2 save

# Mở cổng
sudo ufw allow 5173


Sau khi chạy xong lệnh trên, web của bạn đã online 24/7.
Để xem nó có đang chạy không:
pm2 status
(Nếu thấy trạng thái là online màu xanh lá là ok)

Khi nào bạn muốn TẮT nó:
pm2 stop lms-frontend


Để xóa hẳn nó khỏi danh sách chạy:
pm2 delete lms-frontend
